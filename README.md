## Description
A simple yet effective stock portfolio tracker that calculates total investment value based on user inputs and predefined stock prices. This command-line application allows users to build a stock portfolio, calculate total investment value, and export reports in both TXT and CSV formats.

## Features
- 📊 **Predefined Stock Prices** - Includes 8 major stocks (AAPL, TSLA, GOOGL, AMZN, MSFT, META, NFLX, NVDA)
- 📝 **User Input** - Add multiple stocks with custom quantities
- 💰 **Investment Calculation** - Automatically calculates individual stock values and total portfolio value
- 📄 **Report Export** - Save portfolio summaries as TXT or CSV files
- 🖥️ **User-Friendly Menu** - Simple numbered menu interface
- ✅ **Input Validation** - Handles invalid stock symbols gracefully

## How It Works
1. User selects from menu options
2. Enters stock symbols and quantities
3. System displays portfolio summary with individual and total values
4. Optionally saves report to file (TXT or CSV)
